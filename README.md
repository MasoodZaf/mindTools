# Utility Gamble

A collection of lightweight, high-performance, single-page utility applications built with HTML, CSS, and vanilla JavaScript.

## Utilities

1. **[CalcHive](calchive.html)**
   A comprehensive suite of free online calculators for finance, health, math, and everyday utility. Calculate EMI, BMI, GPA, age, and more with zero signup.
2. **[WordKit](wordkit.html)**
   The ultimate free toolkit for word games. Features a word unscrambler, anagram finder, rhyme dictionary, Wordle helper, crossword solver, and Scrabble tools.

## Development
These applications are entirely static. To view them, simply open the `.html` files in any web browser or serve them using a local HTTP server:
```bash
python3 -m http.server 8080
```
